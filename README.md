# Apollo

The Apollo connector gives Cinatra agents access to Apollo's People API Search — find and verify professional contacts by company domain or name and get back verified email addresses, job titles, and LinkedIn URLs. Full documentation lives in the Integrations hub at https://docs.cinatra.ai/integrations/apollo/

## Works with

- Cinatra agents that search for people or contacts (prospecting, company-discovery, contact-discovery)
- The Ross Index enrichment pipeline (Apollo enriches company and founder records when connected)

## Capabilities

- Search Apollo for people at a target company by domain or company name
- Narrow a people search by job title or location
- Return verified email addresses and LinkedIn URLs for matched contacts
- Emit usage telemetry per search so workspace API consumption is tracked
- Expose connection status and logging settings through the Cinatra configuration UI
